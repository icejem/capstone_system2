## Glossary

**2FA** - Two-Factor Authentication (verify via email)
**SMTP** - Simple Mail Transfer Protocol (email sending)
**SHA-256** - Secure Hash Algorithm (token hashing)
**Rate Limiting** - Preventing too many requests
**Token** - Unique code for verification
**Hashing** - One-way encryption (cannot reverse)
**Throttle** - Temporary rate limit
**Middleware** - Layer that processes requests
**Mailable** - Laravel's email class
**Model** - Database representation in code
**Controller** - Handles business logic
**Route** - URL mapping
**View** - HTML template
